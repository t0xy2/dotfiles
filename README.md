# dotfiles

my current personal configurations i use on my X250 with arch linux

## Contents
- `bspwm` – Window manager config
- `sxhkd` – Hotkeys
- `picom` – Compositor
- `polybar` – Status bar
- `rofi` – App launcher
- `alacritty` – Terminal config
  
- `.zshrc` – Shell config
- `.xinitrc` – X session startup

- `walls` – some of my wallpapers i use
